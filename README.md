## 🖤 The Vibe: Gothic Tech & Transparency

**DeadlineApp** is not just another task manager. It's a statement. Inspired by a **'Gothic Tech'** aesthetic, it combines elegant transparency with sharp, dark visuals. Whether you are in Light or Dark mode, the interface adapts seamlessly, keeping you focused on what truly matters: your ticking clock.

Featuring the iconic **'Azrail & Hourglass'** animation, it turns your deadlines into a visual experience, reminding you that every second counts.

---

## 🚀 Key Features

- ✍️ **Advanced Note-Taking**: Fully integrated **Quill Editor** for rich-text note management.
- ✅ **Task Management**: Streamlined To-Do lists that keep you on top of your game.
- ⏳ **Animated Deadlines**: A flagship hourglass animation that visualizes your remaining time.
- 🌑 **Adaptive Design**: Native support for Dark & Light modes with a premium glassmorphism finish.
- 🍅 **Pomodoro Timer** *(Coming Soon)*: Stay productive with integrated deep-work sessions.

## 🛠️ Installation

Get the project up and running locally in minutes.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hkursatakburak/DeadlineApp.git
   cd DeadlineApp
   ```

2. **Get dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the application:**
   ```bash
   flutter run
   ```
